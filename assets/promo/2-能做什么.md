# 它能做什么

AI 出的稿——网页 demo、Markdown 方案、报告——你想发给客户、同事、小圈子,而不是发到全世界。

常规托管都在抢「发布到公网」。HSpace 反着来:**定向分享**。一句话记住它——

> **稿出即递,点开即读,心里有数。**

## 三件事

- 🚀 **稿出即递**:编辑器或 AI 对话里一键,链接+密码一次粘贴发走,零注册零配置。
- 🎯 **心里有数**:谁看了、看了几次有回执;发错了改密码即撤回;每人一链,踢一个人不用换所有人的密码。
- 🔁 **稿在改,链接不变**:内容随迭代更新,历史可回滚,不用再发「最终版_v3」。

## 还有

| 能力 | 说明 |
|---|---|
| 📝 点开即读 | Markdown 自动排成阅读页,HTML 原样能跑,对方什么都不用装 |
| 📚 文档合集 | 一批稿打包成一个链接、一个目录(你正在体验) |
| 🤖 AI 可直接调用 | MCP 让 Claude / Cursor 在对话里直接发;OpenAPI 接入 agent |
