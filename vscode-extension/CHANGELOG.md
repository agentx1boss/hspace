# Changelog

## 0.4.2 — 2026-07-04

- 文案更新:定位收窄至 AI 编程开发者,「私域分发」统一改为「定向分享」(功能无变化)

## 0.4.1 — 2026-07-04

- 修复:当 `hspace.apiBaseUrl` 为空(常见于旧版本设置迁移后)时发布报错「Failed to parse URL from /publish」;现自动回退到官方实例

## 0.4.0 — 2026-07-04

- **内容版本化**:链接不变,内容随迭代更新
  - 面板右键页面 →「更新内容(升版)」:单页用当前编辑器文件、合集重选文件夹整组替换,链接与密码不变
  - 「版本历史 / 回滚」:查看历次版本,一键回滚到任意旧版
  - 接收方在阅读页/目录页看到"更新于 <日期>";面板显示当前版本号
  - 合集更新即"篇目增删改";匿名也可更新自己的页面(每次重新安全扫描)

## 0.3.0 — 2026-07-04

- **每人一链 / 多口令**:面板右键页面 →「管理访问人」,为每个接收者生成独立密码
  - 按人统计访问量与最近访问时间(👁 N · 最近 …)
  - 单独撤销某人的访问,其他人不受影响、无需换密码;撤销即时生效(持旧 Cookie 也被挡)
  - 同一个链接,每人一个密码——私域分发的完全体

## 0.2.2 — 2026-07-04

- **访问回执**:「最近发布」面板点刷新即可看到每个链接的累计访问量(👁 N),合集为目录+各篇总和;悬停查看更新时间
- 密码门视觉精修:品牌标记 + 亮暗自适应 + 移动端优化 + 错误抖动(接收方侧)

## 0.2.1 — 2026-07-04

- 合集中的 HTML 篇目新增悬浮「← 目录」按钮,读者不必靠浏览器后退返回目录(仅注入一个自包含元素,不改动页面结构;单独发布的 HTML 页面不受影响)

## 0.2.0 — 2026-07-03

- **文档合集**:右键文件夹或多选文件 →「HSpace: 发布为合集」,把一批 md/html 打包成**一个链接 + 一个密码 + 一个目录页**
  - 上传前弹出确认清单(默认全选、显示每篇标题预览),隐私优先绝不静默上传
  - 接收方输一次密码通览全集;目录页 + 逐篇阅读,md 篇目带宽屏侧栏与上/下篇导航
  - 深链保留:转发的「第 N 篇」链接,输完密码直达该篇
  - 「最近发布」面板中合集可展开查看篇目
- 合集 MVP 篇目不可变(改密码/删除可用),增删改内容并入后续版本化

## 0.1.4 — 2026-07-03

- **Markdown 私密分享**:发布 `.md` / `.markdown` 文件,自动渲染成排版精良的阅读页(标题 / 表格 / 代码块 / 引用,亮暗双主题,首个 `#` 标题自动成为页面标题)
- md 原文存储、访问时边缘渲染:阅读模板升级对已发布内容即时生效
- 发布入口文案改为「发布当前文件」,右键菜单与云图标对 md 文件同样可用

## 0.1.3 — 2026-07-03

- **全新品牌视觉**:橙色 appicon 图标 + 墨色横幅
- **品牌统一 HSpace**:命令、设置、视图前缀由 `htmlshare.*` 改为 `hspace.*`(旧版本手动填过的设置需重填)
- **产品定位更新**:README 与商店描述改为「私密分享 AI 生成的内容」——链接+密码,只给该看的人
- 配套后端加固:匿名 TTL 钳制、密码防暴力破解、日配额、体积/访问量/钓鱼特征限制

## 0.1.x

首个公开版本。
- **发布强制带密码**:每次发布自动生成随机 4 位数字密码,通知中直接显示
- 剪贴板改为复制「链接 + 密码」,可直接粘贴分享
- 暂时屏蔽 `hspace.alwaysAskPassword` 配置项(询问逻辑移除,后续版本可能恢复)
- 插件更名为 **HSpace**(ID `agentx1boss.hspace`),publisher 改为 `agentx1boss`
- 品牌统一为 HSpace:命令、设置、视图前缀由 `htmlshare.*` 改为 `hspace.*`(旧版本的自定义设置需重新填写)
- README 拆分中英双语(README.md 英文 / README.zh-CN.md 中文)
- 新增 CI:打 tag 自动发布到 VS Code Marketplace 与 Open VSX
- 一键发布当前 HTML 文件,链接自动复制
- 「最近发布」侧栏面板:打开 / 复制链接 / 设密码 / 删除
- 匿名发布默认 7 天过期,可配置
- 可选 API Key 登录(永久链接)与自建后端支持
