# HSpace

[English](https://github.com/agentx1boss/hspace/blob/main/vscode-extension/README.md) | 简体中文

**给用 AI 编程的你:demo 写完,只发给该看的人。** Cursor / Claude Code 刚生成的 HTML demo、Markdown 方案,一键变成"链接 + 密码"——谁看了有回执,发错了随时撤回,链接不变可迭代。

AI 生成的 demo、报告、方案、数据可视化——你想发给客户、同事、小圈子,而不是发到全世界。HSpace 不是托管,是**定向分享**:每个链接都带密码,不被搜索引擎收录,不怕转发扩散,随时可撤回。

## 为什么不一样

- 📝 **Markdown 即刻成刊**:`.md` 文件自动渲染成排版精良的阅读页(标题、表格、代码块、亮暗双主题)
- 📚 **文件夹成册**:右键文件夹或多选文件 →「发布为合集」,一批 md/html 变成一个链接+密码+目录页,篇间自由导航
- 🔐 **默认私密**:每次发布自动生成随机 4 位密码,没有密码谁也看不到;对方输一次,24 小时免重输
- 📋 **一次粘贴就发走**:链接+密码一起进剪贴板,微信/钉钉/邮件里直接粘
- 🕒 **版本化**:链接不变,内容随迭代更新;查看历史、一键回滚
- 👥 **每人一链**:给每个接收者一个独立密码,按人看访问量,踢一个人不用换所有人的密码
- 👁 **访问回执**:面板刷新即可看到每个链接被打开了多少次
- 🎯 **可控可撤回**:随时改密码、删页面,链接立即失效;到期自动清理,不留痕迹
- 🗂 **发布即管理**:侧栏「HSpace · 最近发布」面板,打开 / 复制 / 改密码 / 删除一目了然
- 🛡 **内容隔离**:每份内容独立子域,互不影响;防暴力破解(连续输错即锁定)

## 快速开始

1. 安装扩展
2. 打开任意 `.html` 或 `.md` 文件
3. 点击编辑器右上角的 ☁️ 图标
4. 链接和密码已复制,粘贴发给该看的人 🎉

无需注册、无需配置,开箱即用。

## 改密码 / 收回访问

- 发布成功的通知里点「修改密码」
- 侧栏「最近发布」面板右键页面 →「设置/修改密码」
- 想收回访问:改个密码(旧密码立即失效),或直接删除页面

## 配置项

| 设置 | 默认值 | 说明 |
|---|---|---|
| `hspace.apiBaseUrl` | 官方托管实例 | 后端 API 地址,可指向自建后端 |
| `hspace.defaultExpiryDays` | `7` | 分享链接的默认有效天数 |

## 使用限制(默认托管实例)

- 仅支持单个 HTML / Markdown 文件,匿名 ≤ 512 KB,登录 ≤ 2 MB
- 每条链接都会过期,没有永久链接:匿名 ≤ 7 天(一次性、不可续),登录 ≤ 30 天/期,到期前可续期
- 登录(免费)解锁每人一链、版本历史/回滚、续期、更大文件与合集;发一稿始终零注册
- 登录:运行 **HSpace: Sign in with GitHub**,浏览器打开 console(hspace.zhanjian.space/console),GitHub 登录后复制 API key 粘贴回编辑器即可。
- 每 IP 每小时最多发布 20 次;匿名另有每天 50 次上限
- 匿名分享必须保持密码保护,发布后不可替换内容(登录后解锁)
- 密码输错限 15 分钟内 10 次,超出暂时锁定
- 匿名页面累计访问超 1 万次后自动下线
- 禁止发布钓鱼、恶意脚本等违规内容,违规页面会被下架

## 自建后端

后端是一个开源的 Cloudflare Worker(R2 + D1 + KV)。部署自己的实例后,把 `hspace.apiBaseUrl` 指过去即可。详见 [GitHub 仓库](https://github.com/agentx1boss/hspace)。

## 隐私说明

发布操作会把当前 HTML 文件内容上传到后端并生成带密码的链接;除此之外不收集任何数据。删除页面后内容立即不可访问,过期页面自动失效。
