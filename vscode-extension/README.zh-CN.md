# HSpace

[English](https://github.com/agentx1boss/hspace/blob/main/vscode-extension/README.md) | 简体中文

**私密分享 AI 生成的内容。** 一键把 AI 帮你写好的 HTML 或 Markdown 发布成"链接 + 密码",只交给该看的人。

AI 生成的 demo、报告、方案、数据可视化——你想发给客户、同事、小圈子,而不是发到全世界。HSpace 不是托管,是**私域分发**:每个链接都带密码,不被搜索引擎收录,不怕转发扩散,随时可撤回。

## 为什么不一样

- 📝 **Markdown 即刻成刊**:`.md` 文件发布后自动渲染成排版精良的阅读页(标题、表格、代码块、亮暗双主题)
- 🔐 **默认私密**:每次发布自动生成随机 4 位密码,没有密码谁也看不到;对方输一次,24 小时免重输
- 📋 **一次粘贴就发走**:链接+密码一起进剪贴板,微信/钉钉/邮件里直接粘
- 🎯 **可控可撤回**:随时改密码、删页面,链接立即失效;到期自动清理,不留痕迹
- 🗂 **发布即管理**:侧栏「HSpace · 最近发布」面板,打开 / 复制 / 改密码 / 删除一目了然
- 🛡 **内容隔离**:每份内容独立子域,互不影响;防暴力破解(连续输错即锁定)

## 快速开始

1. 安装扩展
2. 打开任意 `.html` 或 `.md` 文件
3. 点击编辑器右上角的 ☁️ 图标
4. 链接和密码已复制,粘贴发给该看的人 🎉

无需注册、无需配置,开箱即用。

## 改密码 / 收回访问

- 发布成功的通知里点「修改密码」
- 侧栏「最近发布」面板右键页面 →「设置/修改密码」
- 想收回访问:改个密码(旧密码立即失效),或直接删除页面

## 配置项

| 设置 | 默认值 | 说明 |
|---|---|---|
| `hspace.apiBaseUrl` | 官方托管实例 | 后端 API 地址,可指向自建后端 |
| `hspace.defaultExpiryDays` | `7` | 分享链接的默认有效天数 |

## 使用限制(默认托管实例)

- 仅支持单个 HTML / Markdown 文件,匿名 ≤ 1 MB,登录 ≤ 2 MB
- 匿名分享最长 7 天有效(登录后可永久)
- 每 IP 每小时最多发布 20 次;匿名另有每天 50 次上限
- 匿名分享必须保持密码保护,发布后不可替换内容(登录后解锁)
- 密码输错限 15 分钟内 10 次,超出暂时锁定
- 匿名页面累计访问超 1 万次后自动下线
- 禁止发布钓鱼、恶意脚本等违规内容,违规页面会被下架

## 自建后端

后端是一个开源的 Cloudflare Worker(R2 + D1 + KV)。部署自己的实例后,把 `hspace.apiBaseUrl` 指过去即可。详见 [GitHub 仓库](https://github.com/agentx1boss/hspace)。

## 隐私说明

发布操作会把当前 HTML 文件内容上传到后端并生成带密码的链接;除此之外不收集任何数据。删除页面后内容立即不可访问,过期页面自动失效。
